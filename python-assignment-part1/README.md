# Part 1 — Student Grade Tracker

## Overview

This program works with student data and performs basic operations like cleaning input, analyzing marks, and generating a class summary. It also includes a small string manipulation task.

The main focus is on using core Python concepts like lists, dictionaries, loops, and conditionals.

---

## Tasks Covered

### Task 1 — Data Cleaning

* Removed extra spaces from names
* Converted names to proper format (title case)
* Converted roll numbers from string to integer
* Converted marks from string to list of integers
* Checked if names are valid (only alphabets)
* Printed student profile cards

---

### Task 2 — Marks Analysis

* Assigned grades based on marks
* Calculated total and average marks
* Found highest and lowest scoring subjects
* Allowed user to add new subjects using input
* Recalculated average after adding marks

---

### Task 3 — Class Summary

* Calculated average marks for each student
* Determined pass/fail status
* Counted total students passed and failed
* Found the class topper
* Calculated overall class average

---

### Task 4 — String Manipulation

* Cleaned a paragraph using `.strip()`
* Converted text to title case
* Counted occurrences of "python"
* Replaced text with "Python 🐍"
* Split paragraph into sentences
* Printed numbered sentences

---

## How to Run

1. Open terminal in this folder
2. Run the script:

```
python part1_grade_tracker.py
```

3. Follow the input prompts for manual entry
